The **selectable** style specifies whether the contents of the element can be selected.

## Syntax

```html
y | yes | n | no
```

## Usage

```html
<split/>
<nu-card selectable="n">
  Non-selectable content.
</nu-card>
<nu-spacer></nu-spacer>
<nu-card selectable="y">
  Selectable content.
</nu-card>
```
