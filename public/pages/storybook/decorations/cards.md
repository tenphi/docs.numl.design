# Cards

## Simple card

```html
<split/>
<nu-card>
  I'm a simple card.
</nu-card>
```

## Clear card

```html
<split/>
<nu-card clear>
  I'm a clear card.
</nu-card>
```

## Fancy card

```html
<split/>
<nu-card
  clear text="b" size="xl" color="special-text" shadow="special"
  image="linear(-15deg, hsi(280), hsi(320))">
  Fancy card
</nu-card>
```
